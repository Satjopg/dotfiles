# dotfiles

## 概要
vim,tmux,bash,brew,atomの設定を管理しています.  
この設定ファイルを使えば私の環境が君の手に...!!
## 環境
OS X El Capitan(10.11.6)
## 使い方
1行書いて、はいおしまい.  

```
$ curl -L raw.github.com/Satjopg/dotfiles/master/install.sh | bash
```
